# Chat & Ideas

*Use this file to track ongoing ideas, discussions, and feature requests for the TV browser.*

### Initial Requirements
- Minimum Android 8 support.
- D-pad support for remote controls.
- Minimalist design.
- Low RAM usage.
- Fluent process.

### Future Ideas
- [ ] Add a "Virtual Mouse" mode for websites that do not support spatial navigation well.
- [ ] Ad-blocking (if it can be done with low memory overhead).
- [ ] Voice search integration using Android TV's built-in microphone button.
