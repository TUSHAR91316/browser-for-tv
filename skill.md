# Skills for TV Browser

This document outlines the core technical skills and approaches used in this project:

1. **Android TV Development**: Using Leanback concepts or customized standard UI to support 10ft UI experiences.
2. **WebView Optimization**: Configuring Android WebView for low-memory environments, enabling hardware acceleration, and optimizing rendering.
3. **Focus Management**: Handling D-Pad key events (`KEYCODE_DPAD_*`) and mapping them to HTML elements to ensure accessibility via remote control.
4. **Performance Tuning**: Using Android Profiler to ensure RAM usage is kept to an absolute minimum and UI rendering remains fluent.
